## Current Decisions

### API Framework
- **FastEndpoints** (v7.2.0) for better performance and developer experience.
- Using `FastEndpoints.Swagger` for OpenAPI specification generation.

### Authentication Architecture
- **IAuthService** interface in Core layer for dependency inversion
- **AuthService** implementation in Infrastructure layer following Clean Architecture
- JWT token generation uses FastEndpoints.Security JwtBearer.CreateToken
- Response types documented in endpoint configuration

### Package Management
- Using Central Package Management (Directory.Packages.props)
- All package versions managed in single location

### OpenAPI/Scalar Configuration
- Scalar points to `/swagger/v1/swagger.json` generated by FastEndpoints.
- Dark mode enabled
- Curl as default HTTP client

### Database
- PostgreSQL 16 running in Docker
- Connection string configured for Docker network
