# Product Context

## Problem Statement
Dhamma teachers need a way to:
1. Conduct interactive sessions with multiple participants
2. Assess participants' understanding of teachings
3. Track individual progress over time
4. Maintain records of attendance and performance

## Solution
A real-time web application that enables:
- Interactive question-and-answer sessions
- Immediate feedback and scoring
- Historical tracking of participation
- Offline operation for retreat settings

## User Experience Goals

### For Participants
- Simple mobile interface for answering questions
- Clear feedback on performance
- Ability to track their learning journey
- Offline access via local network

### For Moderators/Admins
- Easy question bank management
- Real-time session control
- Visual presentation mode for projectors
- Comprehensive audit logs

## Key Differentiators
1. **Offline-First**: Works without internet on local network
2. **Real-time**: Instant question push and answer collection
3. **Multiple Attempts**: Encourages learning through retry
4. **Presentation Mode**: Optimized for group viewing
5. **Audit Trail**: Complete history for analysis

## Use Cases

### Use Case 1: Dhamma Talk with Q&A
1. Teacher conducts talk on Four Noble Truths
2. Pushes questions to participants' phones
3. Participants answer on their devices
4. Results displayed on projector

### Use Case 2: Assessment Session
1. Multiple questions on a topic
2. Participants have 3 attempts per question
3. Points awarded based on attempt number
4. Final scores recorded

### Use Case 3: Progress Tracking
1. Participant attends multiple sessions over months
2. System tracks cumulative scores
3. Identifies areas needing more study

## Constraints
- Must work without internet (LAN only)
- Mobile-responsive for participant phones
- Tablet-optimized for admin interface
- Docker deployment for easy setup
